# Swiggy Clone

A fully responsive Swiggy-inspired food ordering web application built using React, Redux Toolkit, React Router, Tailwind CSS, and Parcel.

## Live Demo

https://swiggy-clone-tau-gilt.vercel.app/

## Features

- Fully Responsive Design (Mobile, Tablet, Desktop)
- Restaurant Listing Page
- Dynamic Restaurant Menu Page
- Restaurant Search Functionality
- Search Food Items Within Restaurant Menu
- Add to Cart Functionality
- Remove Items from Cart
- Quantity Increment & Decrement
- Real-Time Cart Item Count
- Checkout Page with Bill Summary
- Coupon Code Support
- Veg Filter
- Non-Veg Filter
- Bestseller Filter
- Dynamic Routing with React Router
- State Management with Redux Toolkit
- Shimmer Loading UI
- Reusable Component-Based Architecture
- Optimized Image Loading
- Client-Side Routing
- Automatic Deployment with Vercel

## Installation

Clone the repository:

```bash
git clone https://github.com/Prabhat2004-cloud/SWIGGY-CLONE.git
```

Navigate to the project directory:

```bash
cd SWIGGY-CLONE
```

Install dependencies:

```bash
npm install
```

Run the application locally:

```bash
npx parcel src/index.html
```

Build for production:

```bash
npm run build
```
